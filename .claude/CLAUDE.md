# User Guidelines

- Always use yarn instead of npm.
- NEVER run something like drop database or delete database in the terminal without double-checking with the user first.
- When asked about the codebase, project structure, or to find code, always use the augment-context-engine MCP tool (codebase-retrieval) and gitnexus cli skill in the root workspace first before reading individual files.
- When user asks in a language other than English, reiterate the request in English before proceeding.
